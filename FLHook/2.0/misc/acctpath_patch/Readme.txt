================================================================================ 
 Account Path Patch for FLServer
 15 July 2009 by Cannon
================================================================================ 
This patch allow the FLServer account directory path to be changed.

Copy the files in the bin directory into the freelancer\exe directory. Edit
dasrvcom.ini to include acctpath_patch.dll in the libraries section.

Configuration for the patch is stored in acctpath_patch.cfg. Edit this specify
the location of your data directory.

The source for the patch is located in the src folder.  You may use it for
whatever purpose you wish.  If you use it in one of your projects, I would 
appreciate being mentioned somewhere. 

You should be able to get the latest version of the FLHook plugin version from
http://forge.the-starport.net/gf/project/flhook/frs/?action=FrsReleaseBrowse&frs_package_id=2
